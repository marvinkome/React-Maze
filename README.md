### Installation & Setup

1.  Clone this repository and install its dependencies.
```bash
$ git clone git://github.com/israeladura/React-Maze.git
$ cd React-Maze
$ yarn install # or npm install
```

3.  From within the directory, start the server.
```bash
$ yarn start
```

4.  Application is live on
 > http://localhost:3000
